            # 在 Ubuntu 安裝 Node.js

            # 前言

            要使用前端框架或各種 npm 套件，要先安裝 Node.js，
            以下紀錄在 Ubuntu 安裝 Node.js 的過程。

            # 參考官網

            使用套件管理器安裝 Node.jsopen in new window
            NodeSource Node.js Binary Distributionsopen in new window

            # 在 Ubuntu 上安装 Curl

            要先安裝 Curl 才能執行後面的步驟。

            sudo apt update
sudo apt install curl

            # 安裝 Node.js

            到官網查詢要安裝的版本號碼，
            目前的長期維護版本為 18.x，
            也可以根據需要安裝更新的版本。

            curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash - &&\
sudo apt-get install -y nodejs

            # 確認版本

            輸入以下指令可以確認安裝是否成功。

            node  -v
