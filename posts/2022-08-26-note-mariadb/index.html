      # MariaDB 修改 innodb_page_size 設定問題

      # 解決方式

      當修改 mariaDB 的設定 conf.d/mysql.cnf
      [mysqld]
      innodb_page_size=32K

      如果重啟失敗
      到 var/lib/mysql (此為預設儲存路徑)
      删除 ib_logfile*
      删除 ibdata*
