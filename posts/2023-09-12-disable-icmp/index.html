          # 在 Ubuntu 設定關閉 ICMP

          # 前言

          當主機不通會使用 ping 指令來測試主機會不會通。
          然而在一些特別的情況下不希望被 ping 到，
          這時候可以關閉 ICMP 。
          像是討人厭的資安稽核！！

          # 開啟設定

          nano /etc/sysctl.conf

          # 修改設定

          net.ipv4.conf.all.accept_redirects = 0
net.ipv6.conf.all.accept_redirects = 0
net.ipv4.icmp_echo_ignore_all = 1

          # 套用設定

          sudo sysctl -p
