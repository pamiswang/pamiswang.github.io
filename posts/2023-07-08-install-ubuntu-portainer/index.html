          # 在 Ubuntu 安裝 Portainer

          免費的輕量化 GUI 工具 Portainer，Portainer 本身也是一個 docker 容器，

          # 先決條件

          • 安裝並運行最新版本的 Docker
          • Portainer 預設通訊埠 9443
          • Docker 必須以 root 身份運行

          # 新增 volume 讓 Portainer 可以儲存資料

          docker volume create portainer_data

          # 下載與安裝 Portainer 容器

          docker run -d -p 8000:8000 -p 9443:9443 --name portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer-ce:latest

          此時 Portainer 伺服器已經安裝完畢，可以透過 docker ps 指令查看執行列表，成功畫面可參考下方範例。

          pamis@pamis-docker:~/Desktop$ sudo docker ps
CONTAINER ID   IMAGE                          COMMAND                  CREATED       STATUS      PORTS                                                                                  NAMES
0cd5f6a4f201   portainer/portainer-ce:latest   "/portainer"   6 seconds ago   Up 6 seconds   0.0.0.0:8000->8000/tcp, :::8000->8000/tcp, 0.0.0.0:9443->9443/tcp, :::9443->9443/tcp, 9000/tcp   portainer

          # 登入 Portainer

          安裝完畢之後可透過瀏覽器輸入網址進入管理介面

          https://localhost:9443

          請將 localhost 替換為相關的 IP ，如果更改過通訊埠則需要調整對應。
          初次進入將會看到 Portainer 服務器的初始設置頁面。
